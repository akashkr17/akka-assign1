##  Akka Assignment

#### Requirements
###### 1.Scala 2.13.5
###### 2.sbt 1.5.5
###### 3. akka 2.6.15

#### Commands
##### clean

This command cleans the sbt project by deleting the target directory. The command output relevant messages.

```bash
sbt clean 
```
##### compile 
This command compiles the scala source classes of the sbt project.
```bash
sbt compile
```

##### tests

```bash
sbt test
```
#### Library dependencies
###### 2.scalafmt
###### 2.scalatest
